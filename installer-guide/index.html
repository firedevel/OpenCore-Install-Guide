# 创建引导U盘

要求:

  • OpenCorePkg (opens new window), 强烈建议运行测试版本来显示更多的测试信息
  • ProperTree (opens new window) 编辑.plist文件(OpenCore Configurator是另一个工具，但已经严重过时，Mackie版本以腐败而闻名。请尽量避免使用这些工具!)
  • 如果您希望使用OpenCore作为主引导加载程序，则必须从系统中完全删除Clover。保留一个基于三叶草的EFI备份。请看这里需要清洁的东西:三叶草转换 (opens new window)

# 在线与离线安装程序

离线安装程序有一个完整的macOS副本，而在线安装程序只有一个恢复映像(~500MB)，一旦启动就从苹果服务器下载macOS。

  • 离线
    • 只能在macOS中制作
    • Windows/Linux没有安装完整安装程序所需的APFS/HFS驱动程序
  • 在线
    • 可以在macOS/Linux/Windows中制作
    • 需要通过目标机器上支持macOS的网络适配器来连接互联网

# 制作安装程序

根据你使用的操作系统，请参阅有关制作USB引导的章节:

  • macOS 用户
    • 支持OS x10.4到当前
    • 支持传统和UEFI安装
  • Windows 用户
    • 支持OS X 10.7到当前
    • 仅限在线安装程序
    • 支持传统和UEFI安装
  • Linux 用户(UEFI)
    • 支持OS X 10.7到当前 *仅限在线安装程序 *适用于支持UEFI 启动的机器
